# **App Name**: ListaNomes PWA

## Core Features:

- Adicionar Nome: Permitir que o usuário insira e adicione novos nomes à lista.
- Visualizar Lista de Nomes: Exibir todos os nomes adicionados em uma lista clara e organizada.
- Editar e Remover Nomes: Funcionalidades para editar nomes existentes ou removê-los da lista.
- Buscar Nomes: Permitir ao usuário procurar nomes específicos na lista.
- Sugestão de Nomes Inteligente: Uma ferramenta de IA que sugere nomes com base em critérios simples fornecidos pelo usuário.
- Funcionalidade PWA: Suporte completo a Progressive Web App (PWA) para instalação na tela inicial e uso offline da lista de nomes.

## Style Guidelines:

- Cor primária: um azul sereno e limpo (#2263DB) para transmitir organização e clareza. Este tom foi escolhido por sua versatilidade e a sensação de calma que evoca.
- Cor de fundo: um azul muito claro, quase branco (#EBF3F8), mantendo o matiz da cor primária, mas altamente dessaturado para otimizar a legibilidade em um esquema claro.
- Cor de destaque: um tom de ciano-verde (#14A3A3), análogo à cor primária, para adicionar um ponto de interesse e contraste sutil.
- Fonte principal para títulos e corpo: 'Inter' (sans-serif) por seu visual moderno, objetivo e alta legibilidade, ideal para listagens e informações textuais.
- Utilize ícones minimalistas e intuitivos (como adicionar, editar, remover, buscar) para facilitar a navegação e a interação com a lista.
- Design responsivo e focado em dispositivos móveis, com um layout de lista limpo e fácil de visualizar, garantindo uma boa experiência de usuário em qualquer tela.
- Animações sutis e funcionais para feedback visual, como ao adicionar ou remover um nome da lista, para aprimorar a experiência do usuário sem distrair.
